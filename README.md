# Fake 3D in C using RayLib

This is a small project that implements a projection logic based
 on the video by Tsoding, namely ![One Formula That Demystifies 3D Graphics](https://www.youtube.com/watch?v=qjWkNZ0SXfo)

At the moment, the only 3D shape implemented is the Cube.
